## SCNU-编译原理实验

### 实验一：Tokenizer

C++源程序分词

### 实验二：RegexToDFA

正则表达式转NFA；NFA转DFA；最小化DFA

### 实验三：TinySyntaxTree

Tiny文法扩充并进行词法分析与语法分析（生成语法树）

### 实验四：LALR

求非终结符的first集合、follow集合；求文法的LR(1)的DFA图、LALR(1)的DFA图、LALR(1)的分析表；
